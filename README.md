**Bottleneck Calculator**

A **bottleneck calculator** is a simple tool that helps users understand if their CPU and GPU are balanced. It identifies which component is limiting system performance so users can make better upgrade decisions.

This project is designed for niche websites, developers, and SEO-focused platforms that want to provide a fast and useful performance analysis tool.
**Project Purpose**

The main goal of this project is to provide a structured and practical solution for detecting hardware limitations in a PC setup.

Instead of guessing which part is slow, this tool gives a clear output based on user input.

It is useful for:

- Gamers checking FPS issues

- PC builders planning upgrades

- Tech websites adding interactive tools

- SEO projects targeting performance-related keywords
- 
**Core Features**
  
**CPU and GPU Bottleneck Detection**
   - Compares processor and graphics card performance
   - Shows which component is weaker
   -

The tool takes basic input from the user and processes it using predefined performance ratios.

 **Input Data**

**Processing Logic**
- Match CPU and GPU benchmarks
